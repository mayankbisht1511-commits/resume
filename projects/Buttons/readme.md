# 🎨 Button Collection

A simple collection of stylish buttons built using **HTML5** and **CSS3**. This project demonstrates different button styles along with smooth hover effects, making it perfect for beginners learning CSS.

live demo
https://stunning-dieffenbachia-e6e750.netlify.app/


## ✨ Features

- 🔵 Solid Button
- ⚪ Outline Button
- 🟢 Pill Button
- 🟠 Accent Button
- ✨ Smooth Hover Animations
- 📱 Responsive Layout
- 🎯 Beginner-Friendly Code

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Button-Collection/
│── index.html
│── style.css
│── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your preferred web browser.
3. Explore the different button styles and hover effects.

## 📸 Preview

The project displays four modern button styles:

- Solid Button
- Outline Button
- Pill Button
- Accent Button

Each button includes a unique hover animation to improve user interaction.

## 📚 Learning Outcomes

This project helps you learn:
- CSS Button Styling
- Hover Effects
- CSS Transitions
- Border Radius
- Flexbox Alignment
- Responsive Design Basics
