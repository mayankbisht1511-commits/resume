# Responsive Navigation Bar

A simple and modern responsive navigation bar built using **HTML5** and **CSS3**. The navigation bar includes a logo, navigation links, and a call-to-action button with hover effects.

live demo
https://stunning-dieffenbachia-e6e750.netlify.app/

## Features
- Responsive design
- Clean and modern UI
- Flexbox layout
- Hover effects on links and button
- External CSS styling
- Mobile-friendly

## Technologies Used
- HTML5
- CSS3

## Project Structure

```
Navigation-Bar/
│── index.html
│── style.css
```

## How to Run
1. Download or clone the project.
2. Open `index.html` in any web browser.
